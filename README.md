# Restaurant Management App

## Project Overview
This Android application was developed as part of the ApexPlanet App Development Internship.

## Features
- Restaurant Login Screen
- Email and Password Authentication
- API Integration using Retrofit
- Dashboard Button
- View Menu Button
- Error Handling

## Technologies Used

- Jetpack Compose
- Retrofit
- Android Studio

- Task 2: UI/UX Design
- Task 3: Backend Integration and API Development
- Task 4: Advanced Features and Testing
